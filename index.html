A programming language is a set of instructions and syntax use to create a software program .
Languages that programmers use to write code can be divide into two main part which are "high-level languages" that is compiled into a "low-level language".
computers hardware can only deal with low-level language but this languages are unreadable for people .

high-level languages

High-level languages are designed to be easy to read and understand.
they allow programmers to write in easy words that can be understood easily.
they use logical words like function, while, if, and else which are used in most programming languages.
they also use symbols like < , > , == ,and != that are used in written the codes.
Examples of high-level languages include C++, Java, Perl,python and PHP.

low-level languages

Low-level languages include assembly and machine languages.
Rarely a programmer may decide to code a basic program in an assembly language to ensure it operates as efficiently as possible.
An assembler can be used to translate the assembly code into machine code.
the machine language is consist of binary code that are understood immediately by CPU.
